## GenderGap Analyzer: Data Analysis of Gender Wage Inequality

## Abstract
This project analyzes gender wage inequality using a public pay gap dataset. The analysis explores differences in salary, bonuses, and employee distribution across pay quartiles to identify gender disparities in organizations.

## Problem Statement
Gender wage inequality remains a major social issue. This project investigates pay gap patterns using data science techniques.

## Dataset
UK Gender Pay Gap Dataset (paygap.csv)

## Methodology
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Result Interpretation

## Tools Used
Python  
Pandas  
Matplotlib  
Seaborn  
Jupyter Notebook

## Results
The analysis shows an average gender pay gap of approximately 14% across organizations.
